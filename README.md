# Python Beginner Programs

This repository contains small Python programs I wrote while learning Python.

## Programs

01 greatest_of_four_numbers.py  
   Finds the greatest number among four user inputs.

02 students_marks_sheet.py
   Calculates total marks and result of a student.

03_simple_calculator.py  
   A simple calculator that performs addition, subtraction, multiplication, and division.

04_simple_guess_game.py  
A number guessing game where the computer generates a random number and the player must guess it.
The program gives hints if the guess is too high or too low until the correct number is guessed.
