# 04_simple_guess_game.py
A number guessing game where the computer generates a random number and the player must guess it.
The program gives hints if the guess is too high or too low until the correct number is guessed.

# 05_simple_spam_filter.py
A simple spam detection program that checks user comments for common spam phrases 
like "buy now", "click this", or "make a lot of money". 
If any spam phrase is detected, the program marks the comment as spam.

# 06_multiplication_table_generator.py

- Beginner friendly Python logic

# 07_factorial_calculator.py

# 08_greatest_number_function.py

# 09_star_pattern.py

# 11_student_marks_analyzer.py
